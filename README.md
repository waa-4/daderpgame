# DaDerpGame v6.2 — Quality of Life Update

## Added

1. Hide/show chat with an unread counter
2. Drawing redo
3. Toggleable minimap
4. Ping Marker tool
5. Automatic and manual AFK status
6. Saved UI scale control
7. Master, music, and effects volume settings

## Modular files

- `systems.js` — shared censor and settings
- `tools.js` — harmless tools and moderation menu
- `qol.js` — chat hiding, redo support UI, minimap, ping markers, AFK, scaling, and volume controls
- `app.js` — main game systems
