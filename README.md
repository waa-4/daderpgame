# DaDerpGame v8.0 — Dual Dimension

## Every game now has two launch buttons

- **Play 2D** uses the original version and blue button.
- **Play 3D** uses the same room, rules, players, drawings, and mode data with a green button and 3D renderer.

3D visualization is included for OG Derp, Evil Boi, Cube Warfare, Free Drawing, Create Mode, MEAT, Machine Game, and Platformer Chaos.

## Input fixes

- Right-click no longer clears held movement.
- Right-click does not rotate the camera or perform an action.
- The browser context menu is still suppressed during gameplay.
- A and D were corrected for camera-relative 3D movement.

## First person

Zooming all the way in switches to first person automatically. The local cube becomes hidden so it does not obstruct the view. Zooming back out returns to third person.

## Shared 3D features

- Player cubes and 3D chat bubbles
- Camera-relative movement
- Left-drag/touch-drag camera
- Mouse wheel and pinch zoom
- Jumping
- Drawings rendered as lines over the ground
- Create blocks, MEAT maze/hunters/barricades, Machine parts/wires, Evil Boi walls, Warfare bases, and Platformer platforms represented in 3D

The original 2D versions remain available and unchanged.
